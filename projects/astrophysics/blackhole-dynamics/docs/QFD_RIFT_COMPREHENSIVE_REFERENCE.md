## Three Fates of Ejected Material

### Reference Frame Distinction (Critical!)

**Key Insight**: Material can cross the Schwarzschild surface at v ~ c **locally** but still fall back because it's drifting slowly in the binary center-of-mass frame.

```
┌─────────────────────────────────────────────────────┐
│  LOCAL FRAME (near BH1):                            │
│    v_local ~ 0.5-0.9c  (crossing r_s!)              │
│    Looks like high-speed escape!                    │
│                                                     │
│  BUT...                                             │
│                                                     │
│  BINARY COM FRAME (what matters for escape):        │
│    v_COM ~ 10⁻³-10⁻²c  (drifting slowly)            │
│    Needs v_COM > v_escape(binary) to truly escape  │
│    Most material: HIGH v_local, LOW v_COM           │
│    → Falls back!                                    │
└─────────────────────────────────────────────────────┘
```

### The Three Outcomes

**Fate 1: True Escape (~1%)**
```
Criteria:
  ✓ Crosses L1 saddle point
  ✓ v_COM > v_escape from total binary mass
  ✓ Not captured by companion BH
  ✓ Reaches infinity

Final state:
  - Far from binary (r → ∞)
  - "Practically drifting" at v << c
  - Lost most KE climbing out of well
  - Energy conservation: E_kinetic(near) + Φ(near) = E_kinetic(far) + Φ(far)

Observable:
  - H/He dominated ejecta (mass-dependent escape)
  - Slow velocities at large distances (~10³-10⁴ km/s)
  - Rare flare events (~1% of crossings)
```

**Fate 2: Capture by Companion (~variable)**
```
Process:
  - Crosses rift from BH1 toward BH2
  - Insufficient v_COM to escape binary
  - Captured by BH2's gravitational well
  - Accretes onto BH2

Implication:
  - If BH2 is a white dwarf → Type Ia supernova!
  - Mass transfer between compact objects
  - Explains some observed SNe Ia rates

Probability:
  - Depends on separation, relative masses
  - Typically few % when rift opens
```

**Fate 3: Falls Back (~99%)**
```
Process:
  - Crosses r_s at v ~ c locally (high-speed!)
  - But v_COM << v_escape in binary frame
  - Ballistic trajectory peaks
  - Returns to BH1

Why so common:
  - Reference frame mismatch!
  - High local velocity ≠ high system velocity
  - Analogy: Jump off train at 100 mph (train frame)
            but train only going 5 mph (ground frame)
            → you land at 5 mph, don't escape Earth!

Observable:
  - Most "flare attempts" are invisible
  - Material recycles through disk
  - This is why flares are RARE!
```
