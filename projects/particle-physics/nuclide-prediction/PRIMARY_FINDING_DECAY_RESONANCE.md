**Question**: Where do decay products land relative to the three curves?

## Statistical Validation

### Chi-Square Test

**Null hypothesis**: Products land randomly relative to curves (5% within ±0.5 Z expected)

**Observed**:
- charge_poor: 270 (9.0%)
- charge_nominal: 308 (10.3%)
- charge_rich: 158 (5.3%)

**Expected (random)**:
- Each curve: ~149 (5%)

**Chi-square calculation**:
```
χ² = Σ (Observed - Expected)² / Expected

χ²_poor = (270 - 149)² / 149 = 98.1
χ²_nom = (308 - 149)² / 149 = 169.6
χ²_rich = (158 - 149)² / 149 = 0.5

χ²_total = 268.2 (for overall landing)
```

**For asymmetric resonance** (β⁻ to poor, β⁺ to rich):
- β⁻ to poor: (240 - 70)² / 70 = 413
- β⁺ to rich: (149 - 71)² / 71 = 86
- **Combined**: χ² ≈ 500+ (using full distribution)

**Critical value** (df=2, p=0.001): 13.8

**Conclusion**: p < 0.00001 (overwhelmingly significant)
