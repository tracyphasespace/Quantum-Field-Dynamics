## Data Regeneration

If `harmonic_scores.parquet` is missing, regenerate:

```bash
# 1. Parse NUBASE2020 (need raw file)
python src/parse_nubase.py --input data/raw/nubase2020_raw.txt

# 2. Fit families
python src/fit_families.py --nuclides data/derived/nuclides_all.parquet

# 3. Score all nuclides
python src/score_harmonics.py \
  --nuclides data/derived/nuclides_all.parquet \
  --params reports/fits/family_params_stable.json \
  --out data/derived/harmonic_scores.parquet
```
