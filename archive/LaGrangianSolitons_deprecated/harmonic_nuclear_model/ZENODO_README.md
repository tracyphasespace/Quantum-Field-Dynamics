## Reproducibility Instructions

### System Requirements

- **Python**: 3.8+
- **RAM**: 4 GB minimum
- **Disk Space**: 100 MB (without data), 500 MB (with data)
- **OS**: Linux, macOS, Windows (WSL)

### Installation

```bash
# 1. Extract archive
unzip harmonic_nuclear_model_v1.0.0.zip
cd harmonic_nuclear_model/

# 2. Install dependencies
pip install -r requirements.txt

# 3. Download NUBASE2020 data
cd data/raw/
wget https://www-nds.iaea.org/amdc/ame2020/nubase_1.mas20.txt
cd ../..

# 4. Run complete pipeline (~10 minutes)
bash scripts/run_all.sh
```

### Expected Outputs

After running the pipeline:
```
data/derived/
  ├── nuclides_parsed.parquet       (3,558 nuclides)
  ├── harmonic_scores.parquet       (All scored)
  └── candidates_by_A.parquet       (Null universe)

reports/
  ├── fits/family_params_stable.json
  ├── exp1/exp1_results.json
  └── tacoma_narrows/tacoma_narrows_results.json
```

### Validation Tests

```bash
# Test 1: Check nuclide count
python3 -c "import pandas as pd; df = pd.read_parquet('data/derived/harmonic_scores.parquet'); print(f'Nuclides: {len(df)}'); assert len(df) == 3558"

# Test 2: Verify A = 161 breakpoint
grep -q '"A_cutoff": 161' reports/tacoma_narrows/tacoma_narrows_results.json && echo "✓ Cutoff verified"

# Test 3: Run two-center validation
python3 src/two_center_model.py --scores data/derived/harmonic_scores.parquet --params reports/fits/family_params_stable.json --out reports/two_center/
```

## Key Findings Summary

### 1. Single-Center Model (A ≤ 161)

**Validates**:
- Shape transition at A = 161 (spherical → deformed)
- Tacoma Narrows effect: Higher ε → longer half-life (r = 0.102, p = 9×10⁻⁵)
- Integer ladder: β⁻ decay parents cluster at integer N (χ² = 873, p ≈ 0)
- Universal dc3: Only 1.38% variation across families

**Physical interpretation**: Resonance with vacuum field drives instability (like Tacoma Narrows Bridge collapse).

### 2. Two-Center Model Extension (A > 161)

**Validates**:
- Dual-core soliton hypothesis: Core bifurcates at A ≈ 161 into prolate ellipsoid
- Correlation recovery: r = 0 → r = +0.34 for heavy nuclei (Δr ≈ +0.35)
- Universal mechanism: Tacoma Narrows effect works for all geometries
- Deformation parameter: β ≈ 0.25 for rare earths/actinides

**Physical interpretation**: Nuclear shape arises from soliton topology, not phenomenological assumptions. Wrong geometry → wrong physics.
